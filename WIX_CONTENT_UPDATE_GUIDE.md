## Step 1: Add Global CSS Theme

### Method: Wix Studio Code Panel

1. In Wix Studio Editor, look for the **Code** icon (`{}`) in the left sidebar or top toolbar
2. Click **"Start Coding"** if this is your first time
3. In the Code panel, click **"Page Code"**
4. Under the **CSS** section, click **"+ Add global.css"**
5. Copy the entire contents of `WIX_GLOBAL_CSS_THEME.css`
6. Paste into the `global.css` file
7. Click **Save**

**OR via Settings:**
1. Click **Settings** (gear icon) in top bar
2. Go to **Advanced** → **Custom Code**
3. Add CSS in **Head Code** section
